# Pizzeria La Dioza

Site vitrine moderne, responsive et gourmand pour **Pizzeria La Dioza** à Roubaix.

Le site présente les pizzas artisanales 100% HALAL, les menus, les tarifs, les zones de livraison, les avis clients et les informations de contact.

## Installation

```bash
npm install
```

## Lancement local

```bash
npm run dev
```

## Build production

```bash
npm run build
```

Le dossier généré est `dist`.

## Aperçu du build

```bash
npm run preview
```

## Déploiement Vercel

1. Pousser le projet sur GitHub.
2. Importer le dépôt dans Vercel.
3. Laisser Vercel détecter Vite.
4. Vérifier les paramètres :
   - Install Command : `npm install`
   - Build Command : `npm run build`
   - Output Directory : `dist`
5. Déployer.

`node_modules`, `dist` et les fichiers `.env` sont ignorés par Git.
